  HOW TO USE (EAS Build)
-----------------------------------------------------------
  1. Place keystore.jks in your project root.
  2. In eas.json, configure:
     {
       "submit": {
         "production": {
           "android": {
             "serviceAccountKeyPath": "./keystore.jks",
             "track": "production"
           }
         }
       },
       "build": {
         "production": {
           "android": {
             "credentialsSource": "local"
           }
         }
       }
     }
  3. In credentials.json:
     {
       "android": {
         "keystore": {
           "keystorePath": "./keystore.jks",
           "keystorePassword": "RubelStore@2024Secure!",
           "keyAlias": "rubel-gemini-ai",
           "keyPassword": "RubelStore@2024Secure!"
         }
       }
     }
  4. Run: eas build --platform android --profile production
